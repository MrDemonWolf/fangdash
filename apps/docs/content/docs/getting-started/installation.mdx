<Callout type="info">
**TL;DR**
- Requires Node.js 20+, pnpm 10.x, a Cloudflare account, and a Twitch app
- Clone → install → create D1 database → copy env files → `pnpm dev`
- Web runs at `localhost:3000`, API at `localhost:8787`
</Callout>

This guide is for developers who want to run FangDash locally or contribute to the project.

## Prerequisites

<Callout type="warn">
You need a **Cloudflare account** (for the D1 database) and a **Twitch Developer application** (for OAuth) before you can run FangDash locally.
</Callout>

| Requirement | Version |
| --- | --- |
| Node.js | 20 or higher |
| pnpm | 10.x (`corepack enable` to install) |
| Cloudflare account | Free tier works |
| Twitch Developer app | [dev.twitch.tv/console](https://dev.twitch.tv/console) |

<Steps>
<Step>
## Clone and Install

```bash
git clone https://github.com/MrDemonWolf/fangdash.git
cd fangdash
pnpm install
```
</Step>

<Step>
## Set Up the Database

Create a Cloudflare D1 database:

```bash
npx wrangler d1 create fangdash-db
```

Copy the database ID from the output and update `apps/api/wrangler.toml`. Then run migrations:

```bash
npx wrangler d1 migrations apply fangdash-db --local
```
</Step>

<Step>
## Configure Environment Variables

Copy the example files from the repo root:

```bash
cp apps/api/.dev.vars.example apps/api/.dev.vars
cp apps/web/.env.example apps/web/.env.local
```

<Callout type="warn">
Never commit `.dev.vars` or `.env.local` — they contain secrets. Both files are already in `.gitignore`.
</Callout>

Fill in `apps/api/.dev.vars`:

```
BETTER_AUTH_SECRET=your-secret-here
BETTER_AUTH_URL=http://localhost:8787
TWITCH_CLIENT_ID=your-twitch-client-id
TWITCH_CLIENT_SECRET=your-twitch-client-secret
```

Fill in `apps/web/.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:8787
```
</Step>

<Step>
## Start Development

```bash
pnpm dev
```

The web app runs at `http://localhost:3000` and the API at `http://localhost:8787`.
</Step>
</Steps>

## Available Scripts

| Command | Description |
| --- | --- |
| `pnpm dev` | Start all apps in development mode |
| `pnpm build` | Build all packages and apps |
| `pnpm test` | Run all tests with Vitest |
| `pnpm type-check` | Type-check all packages |
| `pnpm lint` | Lint all packages |
| `pnpm clean` | Remove all build artifacts |
