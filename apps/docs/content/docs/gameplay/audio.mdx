<Callout type="info">
	**TL;DR** - 3 BGM tracks auto-crossfade based on where you are (menu, solo, race) - 9 sound
	effects covering all major game events - BGM and SFX volume are saved to `localStorage` and
	persist across sessions
</Callout>

FangDash features a full audio system with background music and sound effects.

## Background Music

Three **BGM tracks** play depending on where you are:

| Track    | When it plays                |
| -------- | ---------------------------- |
| Menu BGM | Main menu and lobby screens  |
| Game BGM | Solo endless runner gameplay |
| Race BGM | Multiplayer race gameplay    |

Music **crossfades automatically** when transitioning between screens. No jarring cuts.

## Sound Effects

| Sound       | Trigger                          |
| ----------- | -------------------------------- |
| Jump        | Player jumps                     |
| Double Jump | Player double-jumps              |
| Hit         | Player collides with an obstacle |
| Game Over   | Run ends                         |
| Milestone   | Distance milestone reached       |
| Countdown   | Race countdown ticks             |
| Achievement | Achievement unlocked             |
| Skin Equip  | New skin equipped                |
| Victory     | Race won                         |

## Volume Controls

Two sliders in the settings:

- **BGM Volume** — Background music level
- **SFX Volume** — Sound effects level

Your preferences are saved to `localStorage` and persist across sessions. Set once, forget about it.
